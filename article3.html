Rigidbody を使った物理演算

Rigidbody を追加すると、オブジェクトに重力や衝突の影響を与えることができます。

Rigidbody の基本

スクリプトでオブジェクトに力を加えるには、以下のように記述します。

            
using UnityEngine;

public class AddForceExample : MonoBehaviour
{
    public Rigidbody rb;

    void Start()
    {
        rb = GetComponent();
        rb.AddForce(Vector3.up * 500);
    }
}
