Unityの基本

Unity では、すべてのオブジェクトは「ゲームオブジェクト」として扱われます。

例えば、Cube（立方体）を作ると、それは「ゲームオブジェクト」になります。

ゲームオブジェクトの操作

スクリプトで移動させるには、以下のように記述します。

            
using UnityEngine;

public class MoveObject : MonoBehaviour
{
    void Update()
    {
        transform.position += new Vector3(0.1f, 0, 0);
    }
}
